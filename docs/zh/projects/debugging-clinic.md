# 调试诊所：七个证据案例

<p class="lesson-lead">调试诊所不奖励最快的视觉补丁，而奖励可复现的症状、有证据支持的原因、最小且有意图的修复，以及回归检查。</p>

## 诊所流程

每个案例都需要提交：

```text
复现步骤：
预期结果：
实际结果：
证据：
假设：
实验：
补丁：
回归检查：
```

除非案例明确证明这些做法确实表达了需求，否则不要使用 `!important`、页面级溢出裁剪、文本固定高度或整文件重写。

## 案例 1：消失的样式表

**症状：** 页面显示浏览器默认样式，尽管 `styles.css` 中已经写有规则。

**初始线索：** HTML 文件位于 `pages/`；样式表位于 `assets/css/`；当前 `href` 是 `assets/css/styles.css`。

**证据目标：** 记录网络请求 URL 和状态，然后从 HTML 文件的位置推导出正确的相对路径。

[打开可运行的案例 1](/zh/starters/debugging-clinic/case-1/pages/)

## 案例 2：永远匹配不到的规则

**症状：** 卡片标题仍是灰色，但代码中存在 `.event-card > h2` 规则。

**初始线索：** DOM 中的结构是 `.event-card > .event-card__body > h2`。

**证据目标：** 检查匹配规则列表和直接子元素关系。

[打开可运行的案例 2](/zh/starters/debugging-clinic/case-2/)

## 案例 3：不断升级的特异性

**症状：** 组件状态类无法改变边框颜色。

```css
#events .card { border-color: gray; }
.card.is-selected { border-color: teal; }
```

**证据目标：** 追踪计算结果，并通过重构降低基础选择器的特异性，而不是继续提高状态规则的特异性。

[打开可运行的案例 3](/zh/starters/debugging-clinic/case-3/)

## 案例 4：内容盒导致的溢出

**症状：** 表单在窄屏上造成页面横向滚动。

```css
.registration-form {
  width: 100%;
  padding-inline: 1rem;
}
```

**初始线索：** 默认盒尺寸计算方式仍然生效。

**证据目标：** 计算内容盒与边框盒尺寸；决定在整个项目还是组件范围内使用 `border-box`。

[打开可运行的案例 4](/zh/starters/debugging-clinic/case-4/)

## 案例 5：无法收缩的 Flex 子项

**症状：** 很长的课程标题把工具栏推出容器。

**初始线索：** 标题是一个 Flex 子项；内容中包含一段很长且无法断行的代码；收缩功能已经启用。

**证据目标：** 检查自动最小尺寸和内容换行。最终补丁应保持代码的可读性。

[打开可运行的案例 5](/zh/starters/debugging-clinic/case-5/)

## 案例 6：被压在下面的菜单

**症状：** 菜单设置了 `z-index: 9999`，却仍然位于下一个区块后面。

**初始线索：** 它的父元素具有 `transform: translateZ(0)` 和 `z-index: 1`；相邻区块属于更高的层叠上下文。

**证据目标：** 绘制祖先层叠上下文树，并完成小范围的层级约定修复。

[打开可运行的案例 6](/zh/starters/debugging-clinic/case-6/)

## 案例 7：不具无障碍性的“修复”

**症状：** 自定义下拉菜单外观正确，也能用鼠标打开，却无法通过键盘操作。

**初始线索：** 触发器是一个带点击处理程序和 ARIA 角色的 `div`；没有键盘行为或焦点管理。

**证据目标：** 评估原生元素替代方案、无障碍名称与状态、焦点顺序和键盘激活。完整修复不能只靠 CSS。

[打开可运行的案例 7](/zh/starters/debugging-clinic/case-7/)

## 评分

| 维度 | 每个案例的分值 |
| --- | ---: |
| 复现说明准确 | 2 |
| 证据支持原因 | 3 |
| 假设能够预测一项观察 | 2 |
| 补丁最小且意图明确 | 2 |
| 回归检查覆盖邻近风险 | 1 |

最终评分更重视诊断过程，而不是视觉修复速度。

## 离场问答

选择一个案例，并解释：

1. 最先想到的诱人修复；
2. 它为什么不完整或存在风险；
3. 哪项浏览器证据改变了你的判断；以及
4. 另一个症状相似、原因却不同的缺陷。
